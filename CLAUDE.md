## Что это
**MSK Lounge** — интерактивное цифровое меню для кальянной (кальян + бар + кухня) в Душанбе,
филиал сети Moscow Hookah Chain. Один самодостаточный `index.html` (весь HTML+CSS+JS внутри,
без сборки). Открывается двойным кликом или с любого хостинга. Основной сценарий — **QR на столе,
гость открывает с телефона**.

- Папка проекта: `C:\Users\admin\OneDrive\Desktop\msk`
- Язык общения с пользователем: **русский**. Пользователь не технический — нужны простые шаги,
  любит «покажи в браузере», открываю результат через `Start-Process <url>`.

## Ключевые места в index.html (где что менять)
- **`MENU`** — массив всех категорий и блюд. Категории по `id`: hot, garnish, salad, soup, snack,
  roll, hotroll, sweetroll, coffee, lemon, shake, juice, smoothie, soft, sauce.
  Поля блюда: `n` название, `w` граммовка/объём, `p` цена, `b` бейдж ('hit'|'prem'|'new'),
  `i` состав [массив], `d` описание, `k` ключевое слово для сток-фото, `img` (ставится через REAL).
  `type`: "cards" (с фото) или "list" (напитки/соусы строкой).
- **`REAL`** — карта реальных фото блюд: точное название → `photos/файл.jpg`. ЕДИНАЯ точка подмены фото блюд.
- `IMG` / `SUSHI` / `DRINKS`(PX) — стоковые фолбэк-фото (TheMealDB/Unsplash/Pexels), перекрываются REAL.
- **`GROUPS`** — группировка меню верхнего уровня: `kitchen` и `drinks` (большие разделители Кухня/Напитки).
- **`REVIEWS`** — отзывы (сейчас ЗАГЛУШКИ: Алишер/Мадина/Рустам), с переводами ru/en/tj.
- **`T`** — все строки интерфейса на ru/en/tj. Новые тексты добавлять во все 3 языка + `data-i18n` в HTML.
- **`ORDER_WHATSAPP` / `ORDER_TELEGRAM`** — куда уходит заказ из корзины (пусто → «показать официанту»).
- **`SITE_URL`** — переопределение цели QR (пусто → берёт `location.href`).
- **footer** (HTML) — телефон/адрес/часы. Сейчас: Душанбе, Нарзикулова 11, 2-этаж; Пн–Вс 12:00–00:00;
  (+992) 176 700 700.  ⚠️ На офсайте mskofficial.ru указано до 02:00 и тел. 330-330-330 —
  пользователь оставил свои данные, не менять без спроса.

## Что реализовано (фичи)
- Hero: ВИДЕО-фон (videos/atmosphere.mp4) + фото-фолбэк (msk-hall.jpg), параллакс, золотое свечение,
  шиммер на слове «Lounge». 3D-кальян УБРАН.
- Sticky-навигация: ссылки на категории + метки-разделители групп (Кухня/Напитки)
- Блок «О нас» + строка преимуществ (🕛 до 00:00, 💨 авторские кальяны, 🛋 VIP-залы, 📍 адрес)
- Галерея зала: лента из 10 реальных фото (scroll-snap) + лайтбокс по клику
- Поиск + фильтры (Все / Хиты / Премиум)
- Витрина: 3 плитки-категории (Кухня/Роллы/Напитки) с реальными фото
- Фирменное блюдо: Томагавк (реальное фото)
- Меню из MENU: карточки (фото, кнопка «+», бейджи) и списки (напитки/соусы)
- Модалка по клику: фото, описание, состав тегами с эмодзи
- Корзина «Мой заказ»: «+» на карточках/строках, плавающая кнопка с суммой, дравер с количеством,
  отправка в WhatsApp/Telegram/«показать официанту»
- QR-код: КАРТИНКОЙ через api.qrserver.com (осн.), фолбэки qrcodejs + quickchart.io, кликабельный,
  под ним текст ссылки. (Раньше был баг — пустой QR; починено переходом на image-рендер.)
- Отзывы: 3 карточки с золотыми звёздами
- Атмосфера: дым на canvas (~1% колечек, остальное обычный дым), мягкая дымка (haze),
  золотые искры-угольки снизу, дым-trail за курсором (60% колечки), зерно (noise), свечение за курсором
- Кнопка фоновой музыки (внизу слева) — нужен audio/ambient.mp3 (без файла прячется)
- Полоса прогресса прокрутки (золотая линия сверху)
- Reveal-на-скролл: блоки ниже hero плавно проявляются (класс `.reveal` → `.inview`)
- Кнопка «наверх», i18n RU/EN/TJ
- Везде уважается `prefers-reduced-motion`; видео/дым на паузе при скрытой вкладке
