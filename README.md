# MSK Lounge — интерактивное цифровое меню

Премиальное цифровое меню для **MSK Lounge** (Moscow Hookah Chain · Душанбе) — кальянная, бар и кухня. Один самодостаточный HTML-файл: всё меню, фото, видео-фон зала, 3 языка (RU/EN/TJ), поиск, фильтры, корзина «Мой заказ», QR-код, галерея зала, отзывы и атмосферные анимации (дым, искры, колечки).

## Запуск локально
Открыть `index.html` двойным кликом, либо поднять простой сервер (чтобы корректно работал QR):

```bash
python -m http.server 8080
```
Затем открыть http://127.0.0.1:8080/

## Публикация (чтобы QR работал у гостей)
Залить папку на любой статичный хостинг — **Netlify**, **Cloudflare Pages** или **GitHub Pages**.
QR-код на странице автоматически ведёт на адрес сайта. После публикации можно вписать постоянный адрес в `SITE_URL` внутри `index.html`.

## Структура
- `index.html` — весь сайт (HTML + CSS + JS в одном файле)
- `photos/` — фото блюд и реальные снимки зала MSK Lounge
- `videos/` — видео-фон атмосферы
- `audio/` — фоновый трек (опционально, `ambient.mp3`)
- `plan.md` — исходное техзадание
- `ФОТО-чеклист.md` — чеклист фото блюд

## Где менять контент (в `index.html`)
- **Меню** — массив `MENU`
- **Реальные фото блюд** — карта `REAL`
- **Телефон/адрес/часы** — блок `footer`
- **Отправка заказа** — `ORDER_WHATSAPP` / `ORDER_TELEGRAM`
- **Адрес сайта для QR** — `SITE_URL`
