# 📸 Чеклист фото для меню MSK Lounge

**Как пользоваться:**
1. Находишь красивое фото блюда в интернете.
2. Сохраняешь его в папку `photos/` под именем из колонки «файл» (напр. `burger-beef.jpg`).
3. Отмечаешь галочкой `[x]`. Можно вместо файла просто прислать мне ссылку — подключу сам.
4. Форматы: `.jpg` / `.jpeg` / `.webp`. Желательно горизонтальные, от 700px по ширине.

Приоритет — блюда с крупными карточками (разделы 1–9). Напитки (10–15) — мелкие превью, можно позже.
